About

This folder is a new homepage prototype built from a different reference structure. Instead of the old table-based homepage, the page uses a cleaner academic layout with a fixed left column for identity and navigation, and a main content column for long-form research information.

My recent work spans segmentation, generation, reconstruction, and discrete representation learning. I am especially interested in ideas that begin as task-specific methods but gradually point toward more general representation questions.

Research

  • Medical image analysis: segmentation, generation, reconstruction
  • Discrete representation learning: VQ-VAE and VQGAN style methods
  • Prior technical exposure: LLM acceleration via GPTQ and QuaRot

A working roadmap for my research is: first solve specific problems, then understand the deeper common structure behind them, and finally move toward robust real-world systems.
